## Get Started in Seconds

</div>

### Clone with Hooks
```
git clone https://github.com/buildwithgitstarter/<TheRepoName>.git --config core.hooksPath=.buildwithgit
```

###  Or Download ZIP
Click the green **Code** button → **Download ZIP**

```
git config core.hooksPath .buildwithgit
```
